# Number Guessing Game (Tcl)

This is a simple command-line nummber guessing game in "Tcl".
The program generates a random number within a user-defined range, and the player must guess the number correctly.

# Overview
This code deomonstrates:
- Basic Tcl syntax and procedure.
- User input handling via standard input
- Random number generation
- Control flow and conditional logic

# To execute the program use
- tclsh random_game.tcl

# Example
Upon execution of the code, it will tell you to

Set the upper limit: 20
Guess a number from 1 to 20: 12
Better luck next time. The random number is 17.
